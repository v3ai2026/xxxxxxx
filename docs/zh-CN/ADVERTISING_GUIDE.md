1. 点击 **"+ 创建广告"** 按钮
2. 弹出 AI 广告创建向导

### 步骤 2: 填写产品信息


## 广告平台配置

### Google Ads 配置

1. 访问 [Google Ads API 中心](https://ads.google.com/home/tools/manager-accounts/)
2. 创建管理员账号
3. 获取开发者令牌
4. 设置 OAuth 2.0 凭证
5. 在 `.env.local` 中配置：

```env
GOOGLE_ADS_DEVELOPER_TOKEN=your_token
GOOGLE_ADS_CLIENT_ID=your_client_id
GOOGLE_ADS_CLIENT_SECRET=your_secret
GOOGLE_ADS_REFRESH_TOKEN=your_refresh_token
GOOGLE_ADS_CUSTOMER_ID=123-456-7890
```

**支持的广告类型:**
- 🔍 搜索广告 - 关键词竞价
- 📱 展示广告 - 图片/横幅广告
- 🛒 购物广告 - 产品 Feed
- 📺 YouTube 视频广告

### Facebook Ads 配置

1. 创建 [Facebook App](https://developers.facebook.com/apps/)
2. 添加 Marketing API 权限
3. 生成访问令牌
4. 在 `.env.local` 中配置：

```env
FACEBOOK_ADS_ACCESS_TOKEN=your_token
FACEBOOK_ADS_APP_ID=your_app_id
FACEBOOK_ADS_APP_SECRET=your_secret
FACEBOOK_ADS_ACCOUNT_ID=act_123456789
```

**支持的广告类型:**
- 📷 图片广告
- 🎬 视频广告
- 🎠 轮播广告
- 📖 即时体验广告

### TikTok Ads 配置

1. 注册 [TikTok for Business](https://ads.tiktok.com/)
2. 创建广告主账号
3. 申请 API 访问权限
4. 在 `.env.local` 中配置：

```env
TIKTOK_ADS_ACCESS_TOKEN=your_token
TIKTOK_ADS_APP_ID=your_app_id
TIKTOK_ADS_SECRET=your_secret
```

### 中国广告平台配置

```env
DOUYIN_ACCESS_TOKEN=your_token
DOUYIN_ADVERTISER_ID=your_id
```

## 受众定向

### 基础定向

#### 地理位置
- 国家
- 省份/州
- 城市
- 邮政编码
- 半径定向（例如：某地点 10 公里内）

#### 人口统计
- 年龄范围：18-65+
- 性别：男性、女性、全部
- 语言
- 教育水平
- 婚姻状况

### 高级定向

#### 兴趣定向
基于用户的兴趣爱好：
- 时尚与美容
- 科技与电子产品
- 旅游
- 健身与运动
- 美食
- ... 300+ 兴趣类别

#### 行为定向
基于用户的在线行为：
- 最近购买行为
- 设备使用情况
- 网站访问历史
- 应用使用情况

#### 自定义受众
- 网站访客（使用 Pixel）
- 客户列表上传
- 应用活动用户

#### 相似受众（Lookalike）
自动寻找与现有客户相似的新用户：
```typescript
targeting: {
  lookalike: {
    source: 'existing_customers',
    similarity: 0.8 // 0.0-1.0，越高越相似
  }
}
```

## 预算管理

### 预算类型

#### 每日预算
系统每天最多花费指定金额。

```typescript
budget: {
  daily: 500, // 每日 500 元
  currency: 'CNY'
}
```

#### 总预算
整个广告活动的总花费上限。

```typescript
budget: {
  total: 15000, // 总共 15000 元
  daily: 500    // 每日最多 500 元
}
```

### 出价策略

#### 目标 CPA（每次转化成本）
系统自动优化出价以达到目标 CPA。

```typescript
bidding: {
  strategy: 'target_cpa',
  targetCPA: 50 // 目标每次转化成本 50 元
}
```

#### 目标 ROAS（广告支出回报率）
优化出价以达到目标 ROAS。

```typescript
bidding: {
  strategy: 'target_roas',
  targetROAS: 4.0 // 目标 ROAS 4.0x（每花 1 元赚 4 元）
}
```

#### 最大化转化
在预算内最大化转化数量。

```typescript
bidding: {
  strategy: 'maximize_conversions'
}
```

### 预算分配建议

**测试阶段（前 7 天）:**
- 每日预算：100-300 元
- 运行多个广告变体
- 收集性能数据

**优化阶段（7-30 天）:**
- 暂停低效广告
- 增加高效广告预算 20-50%
- 测试新的受众和创意

**规模化阶段（30+ 天）:**
- 大幅增加高效广告预算
- 扩展到新的平台和受众
- 持续 A/B 测试

### 指标说明

| 指标 | 说明 | 理想值 |
|------|------|--------|
| **展示量** | 广告被看到的次数 | 越高越好 |
| **点击率 (CTR)** | 点击数 / 展示数 | 2-5% |
| **点击成本 (CPC)** | 总花费 / 点击数 | 行业相关 |

| **ROAS** | 收入 / 广告花费 | > 3.0x |

## 自动优化

### AI 智能建议

系统实时分析所有广告，提供智能建议：

```
🤖 AI 智能建议 (5)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ 高优先级:
  • Google Ads "夏季促销" CPA 过高 (¥85)
    建议: 暂停此广告或降低出价
    
💡 机会:
  • Facebook "新品推广" ROAS 高达 5.2x
    建议: 增加预算 30% 以获得更多转化
    
  • TikTok "品牌推广" CTR 3.8%，表现优秀
    建议: 创建相似受众扩大投放

🔄 优化建议:
  • 抖音广告在晚上 8-10 点表现最佳
    建议: 调整投放时段
    
  • 文案 A 比文案 B 点击率高 40%
    建议: 暂停文案 B，专注文案 A
```

### 自动优化规则

系统自动执行以下优化动作：

#### 1. 自动暂停低效广告
**触发条件:**
- CPA > 目标 CPA 的 150%
- 连续 3 天无转化且花费 > ¥500
- CTR < 0.5%

**动作:**
- 自动暂停广告
- 发送通知给你

#### 2. 自动增加高效广告预算
**触发条件:**
- ROAS > 4.0x
- CTR > 3%
- 转化率 > 5%

**动作:**
- 增加预算 20%
- 创建相似受众

#### 3. 自动调整出价
**触发条件:**
- CPA 持续高于目标
- 展示量不足

**动作:**
- 降低/提高出价 10-20%
- 测试不同出价策略

### 手动执行优化

点击 **"🤖 AI 自动优化"** 按钮，系统将：
1. 分析所有广告性能
2. 识别优化机会
3. 执行优化动作
4. 生成优化报告

## 最佳实践

### 1. 测试策略

**A/B 测试:**
- 同时运行 2-3 个广告变体
- 只改变一个变量（标题、图片或 CTA）
- 运行至少 7 天收集数据
- 保留表现最佳的版本

**多变量测试:**
- 测试多个元素的组合
- 需要更大的预算和流量
- 使用 AI 自动识别最佳组合

### 2. 受众策略

**漏斗策略:**
1. **意识阶段** - 广泛受众，品牌曝光
2. **考虑阶段** - 兴趣受众，产品介绍
3. **转化阶段** - 网站访客，再营销

**分层投放:**
- 冷受众 - 降低出价，增加频次
- 温受众 - 中等出价，展示优惠
- 热受众 - 提高出价，强调 CTA

### 3. 预算策略

**7-30-90 规则:**
- 前 7 天：测试阶段，小预算
- 7-30 天：优化阶段，逐步增加
- 30-90 天：规模化阶段，大幅增加

**20% 规则:**
每次预算调整不超过 20%，避免算法重置。

### 4. 创意策略

**3 秒规则:**
前 3 秒吸引注意力，使用：
- 鲜艳颜色
- 动态元素
- 意外效果

**移动优先:**
- 竖屏视频（9:16）
- 大字体文字
- 简洁的设计

### 5. 优化频率

**每日检查:**
- 预算使用情况
- 严重问题（CPA 超高）

**每周优化:**
- 分析性能数据
- 暂停低效广告
- 增加高效广告预算
- 测试新创意

**每月回顾:**
- 生成详细报告
- 调整整体策略
- 规划下月预算

### 6. 常见错误

❌ **避免:**
- 频繁大幅调整预算
- 过早暂停广告（< 7 天）
- 只关注点击率，忽视转化
- 使用单一平台
- 忽视移动端优化

✅ **推荐:**
- 保持耐心，给算法学习时间
- 关注 ROI 而非单一指标
- 多平台投放分散风险
- 持续测试和优化
- 使用 AI 自动化工具

## 疑难解答

### 广告未显示

**可能原因:**
1. 审核未通过
2. 出价太低
3. 受众太窄
4. 预算不足

**解决方案:**
- 检查广告状态和审核反馈
- 提高出价 20%
- 扩大受众范围
- 增加每日预算

### 点击率低

**可能原因:**
- 创意不吸引人
- 受众不精准
- 广告疲劳

**解决方案:**
- 使用 AI 生成新创意
- 优化受众定向
- 更换广告素材

### 转化成本高

**可能原因:**
- 着陆页体验差
- 产品定价问题
- 受众不匹配

**解决方案:**
- 优化着陆页
- 测试不同价格点
- 使用相似受众
